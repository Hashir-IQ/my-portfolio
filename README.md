# Hashir Iqbal — Dark & Techy Portfolio

A single-page scroll portfolio with a dark hacker/developer aesthetic — dark background, neon cyan glowing accents, code-inspired fonts, and smooth scroll animations.

## Technologies Used
- **Vite** + **React**
- **TypeScript**
- **shadcn-ui**
- **Tailwind CSS**
- **Framer Motion**

## Features
- **Hero Section**: Profile picture with glowing cyan border, typing tagline, and contact CTAs.
- **About Me**: Brief intro covering web development, cybersecurity, AI/ML, and Flutter.
- **Skills**: Glow-styled card groups: Web Dev, Programming, Cybersecurity, etc.
- **Timeline**: Experience timeline detailing internships and freelance work.
- **Projects**: Grid display for GPMS, MEDICARE++, and LitUp Solar.
- **Contact Form**: Neon cyan styled contact form.

## Getting Started

Follow these steps to run the project locally:

```sh
# Install dependencies
npm install

# Start the local development server
npm run dev
```
